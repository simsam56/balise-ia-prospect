## Points d'inspiration utilises pour Balise-IA Prospect

1. Atomic CRM:
- approche UI composable,
- import/export natif,
- extensibilite via API.

2. Twenty:
- ergonomie "modern CRM",
- logique pipeline + statut lisible,
- fort focus open-source.

3. Odoo CRM:
- workflow commercial structure,
- suivi d'activites orientes action.

## Positionnement de ce projet

Ce projet est volontairement plus leger et local-first:
- Next.js + SQLite + Prisma,
- demarrage en quelques commandes sur MacBook,
- adapte au workflow solo de prospection B2B Balise-IA.
